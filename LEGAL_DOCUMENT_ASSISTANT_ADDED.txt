STEP 2: Install System Libraries (IMPORTANT!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OCR requires system libraries. Choose your operating system:

WINDOWS:
────────
1. Download Tesseract:
   https://github.com/UB-Mannheim/tesseract/wiki
   Click "tesseract-ocr-w64-setup-v5.x.exe"

2. Run installer
   Accept default installation location
   Complete the installation

3. Download Poppler (for PDF support):
   https://blog.alivate.com.au/poppler-windows/
   Download the Windows version

4. Extract Poppler
   Extract to a location like C:\Program Files\poppler
   (or add to PATH if you know how)

5. Test installation:
   python -c "import pytesseract; print('✓ Tesseract installed')"

If error, check that Tesseract is in PATH or set path in code.

MAC:
────
Run in Terminal:
    brew install tesseract poppler
    pip install pytesseract pdf2image

Verify:
    python -c "import pytesseract; print('✓ Tesseract installed')"

LINUX (Ubuntu/Debian):
──────────────────────
Run in Terminal:
    sudo apt-get install tesseract-ocr poppler-utils
    pip install pytesseract pdf2image

Verify:
    python -c "import pytesseract; print('✓ Tesseract installed')"

STEP 3: Start the App
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    streamlit run civicshield_pro_app.py

New "📄 Legal Documents" page appears in sidebar!
