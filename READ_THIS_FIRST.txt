                      🚀 HOW TO GET STARTED
================================================================================

STEP 1: Install Python Packages
  ➜ pip install -r requirements.txt

STEP 2: Install System Libraries (REQUIRED!)

  WINDOWS:
  --------
  1. Download Tesseract:
     https://github.com/UB-Mannheim/tesseract/wiki
  2. Download Poppler:
     https://blog.alivate.com.au/poppler-windows/
  3. Install both to default locations
  
  MAC:
  ----
  brew install tesseract poppler
  
  LINUX:
  ------
  sudo apt-get install tesseract-ocr poppler-utils

STEP 3: Run the App
  ➜ streamlit run civicshield_pro.py

STEP 4: Try the New Feature
  • Click "📄 Legal Documents" in sidebar
  • Upload a document or take a photo
  • Watch OCR extract text automatically!
